TWO CONNECTED SKILLS

从一篇 PDF，到一条可以检查的论证线

01

PaperFlow Atlas

保存 PDF 原文，分开生成译稿、核心解析和精读笔记。概念、理论、方法与观点可以跨文献连接，并沿页码回到证据。

02

PaperFlow Thread

从问题或关键词组合出发，只检索你自己的 Obsidian 库。它梳理定义、机制、分歧、边界和证据空白，与你共同形成论证。

START ANYWHERE

你可以从现在最需要的一步开始

  • 读懂一篇文献辨认研究问题、主张、方法与证据
  • 积累知识网络让不同文献中的概念和观点彼此连接
  • 准备文献综述把分散材料组织成层层相扣的脉络
  • 推进研究项目保留版本，按你的决定整合新文献

OBSIDIAN WORKFLOW

阅读痕迹和研究结构，留在同一个地方

原始 PDF 会复制到 Obsidian。划词译文用于即时理解，长期批注则保留原文、个人 Note 和页码链接。删除下载目录中的文件后，库内副本仍然可用。
